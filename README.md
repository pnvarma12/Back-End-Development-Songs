# Back-End-Development-Songs

This repository contains the Songs microservice for the Cloud Application Development Capstone project.

## Environment Setup
The development environment for this microservice was established using the following steps:
* **Template:** Created from the provided course repository template.
* **Initialization:** Successfully executed the `bin/setup.sh` script to install all necessary dependencies.
* **Python Version:** Configured to use **Python 3.9.x**.
* **Virtual Environment:** Initialized and activated a virtual environment named `backend-songs-venv`.

## API Documentation
This service provides endpoints for managing song metadata, including health checks and CRUD operations for song records.
